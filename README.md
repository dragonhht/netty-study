# Netty学习

## 1、简介

> Netty是一个NIO客户端服务器框架，可以快速简单地开发协议服务器和客户端等网络应用程序。它极大地简化TCP和UDP套接字服务器等网络编程。

## 2、使用

-   maven依赖

```
<dependency>
    <groupId>io.netty</groupId>
    <artifactId>netty-all</artifactId>
    <version>4.1.20.Final</version>
</dependency>
```
